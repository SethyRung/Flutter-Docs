Ready to build beautiful, multiplatform apps from a single codebase? Choose from the following two options to get started setting up your Flutter development environment.

::card{title="Quick start" icon="i-lucide-rocket" to="https://docs.flutter.dev/get-started/quick" target="\_blank"}
Use VS Code or another Code OSS-based editor to quickly install, set up, and try out Flutter development!
::

::card{title="Custom setup" icon="i-lucide-sliders-horizontal" to="https://docs.flutter.dev/get-started/custom" target="\_blank"}
Install the Flutter SDK, set up any initial target platform, and get started learning and developing with Flutter!
::

::card{title="Developing in China" icon="i-lucide-info"}
If you want to use Flutter in China, check out [using Flutter in China](https://docs.flutter.dev/community/china). If you're not developing in China, ignore this notice and follow the other instructions on this page.

如果你正在中国的网络环境下配置 Flutter， 请参考 [在中国网络环境下使用](https://docs.flutter.cn/community/china/) Flutter 文档.
