## Update Flutter

If you already have Flutter installed and would like to upgrade your Flutter SDK installation or change to a different release channel, check out [Upgrading Flutter](https://docs.flutter.dev/install/upgrade).

When upgrading, also remember to check out the published list of [breaking changes and migration guides](https://docs.flutter.dev/release/breaking-changes).

## Download previous releases

If you're looking to download and install previous releases of Flutter, check out the [SDK archive](https://docs.flutter.dev/install/archive).

::UAlert{icon="i-lucide-info" title="Note" description="We recommend keeping your apps and development environments up to date with the latest stable or beta releases." variant="subtle"}
::
